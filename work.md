# Table Control System 项目规划

## 项目基础信息
- 项目名称: table_control
- 数据库配置: mysql
- 技术栈: Flask + Bootstrap + SQLAlchemy + pandas + openpyxl

## 核心功能模块

### 1. Excel上传与表转换
- 前端：带进度提示的上传表单，支持拖拽上传
- 后端：解析Excel文件，按规则插入到数据库表中

### 2. 指定字段抽新表
- 前端：选择源表→多选字段→自定义过滤规则→输入新表名→提交
- 后端：根据规则创建新表并插入数据

### 3. SQL执行页面
- 前端：SQL编辑器(语法高亮)+执行按钮+结果展示区(分页)
- 后端：执行SQL语句(SELECT/INSERT/CREATE TABLE)，返回结果

### 4. SQL结果存新表
- 前端：在SQL执行结果页面增加"存为新表"按钮
- 后端：将结果存储为新表

## 环境和包管理要求

- 使用 `.env` 环境
- 使用 `uv` 命令管理所有包，替代系统自带的 Python 环境和 pip 命令
- 不使用系统自带的 Python 环境
- 使用 `uv run` 命令替代 python 命令运行应用
